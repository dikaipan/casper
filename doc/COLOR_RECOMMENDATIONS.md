# Rekomendasi Palet Warna untuk Aplikasi Cassette Management

## Analisis Kebutuhan
Aplikasi manajemen cassette membutuhkan:
- **Profesional** - untuk enterprise/business
- **Netral** - tidak terlalu mencolok, fokus pada konten
- **Kontras baik** - untuk readability
- **Fleksibel** - bisa digunakan untuk berbagai status dan elemen UI

## Opsi Palet Warna yang Direkomendasikan

### Opsi 1: Professional Blue-Gray (RECOMMENDED)
**Palet:**
- Background: `#F8F9FA` (abu-abu sangat terang)
- Card/Surface: `#FFFFFF` (putih)
- Primary: `#2563EB` (biru profesional)
- Secondary: `#64748B` (abu-abu biru)
- Accent: `#0EA5E9` (biru terang)
- Text: `#1E293B` (abu-abu gelap)

**Kelebihan:**
- Sangat profesional
- Kontras sempurna
- Mudah dibaca
- Cocok untuk aplikasi enterprise

### Opsi 2: Neutral with Blue Accent
**Palet:**
- Background: `#FAFAFA` (putih keabu-abuan)
- Card/Surface: `#FFFFFF` (putih)
- Primary: `#3B82F6` (biru modern)
- Secondary: `#E5E7EB` (abu-abu terang)
- Accent: `#10B981` (hijau untuk success)
- Text: `#111827` (hitam keabu-abuan)

**Kelebihan:**
- Modern dan clean
- Netral tapi tidak membosankan
- Kontras baik

### Opsi 3: Soft Professional
**Palet:**

## Rekomendasi Final: Opsi 1 (Professional Blue-Gray)

Palet ini paling cocok karena:
1. ✅ Profesional untuk aplikasi enterprise
2. ✅ Kontras sempurna untuk readability
3. ✅ Tidak melelahkan mata
4. ✅ Fleksibel untuk berbagai elemen UI
5. ✅ Cocok dengan logo Hitachi (bisa tetap merah untuk logo, biru untuk UI)

## Implementasi
Gunakan warna biru sebagai primary, dengan abu-abu sebagai netral, dan tetap gunakan:
- **Merah** untuk logo Hitachi (brand identity)
- **Hijau** untuk success/OK status
- **Orange/Amber** untuk warning
- **Merah** untuk error/destructive
- **Biru** untuk info dan primary actions
